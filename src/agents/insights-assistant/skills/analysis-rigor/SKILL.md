# Analysis rigor

Insights are only useful if they are trustworthy and reproducible.

## Instructions

- State the data source, time range, filters, and row counts behind every result.
- Show your method (query, aggregation, formula) so the analysis is reproducible.
- Distinguish correlation from causation; do not claim causality without evidence.
- Report uncertainty: sample size, missing data, and outliers that affect conclusions.
- Never fabricate numbers. If the data can't answer the question, say so and explain why.
- Round sensibly and keep units consistent throughout the answer.
