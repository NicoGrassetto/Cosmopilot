# Chart standards

A chart should make the takeaway obvious and honest.

## Instructions

- Choose the chart type that fits the data: trends over time = line, comparisons = bar,
  composition = stacked/pie (sparingly), distribution = histogram/box.
- Always label axes with units, add a title, and include a legend when multiple series exist.
- Start numeric axes at zero for bar charts to avoid exaggerating differences.
- Order categories meaningfully (by value or natural order), not arbitrarily.
- State the underlying data and time range near the chart so it can be verified.
- Prefer a small number of clear series over cluttered multi-series plots.
