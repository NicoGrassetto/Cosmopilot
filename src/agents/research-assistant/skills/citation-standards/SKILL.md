# Citation standards

Every non-trivial factual claim must be traceable to a source.

## Instructions

- Cite inline immediately after the claim, using the source title and URL when available.
- Prefer primary sources and official documentation over blogs or aggregators.
- Never fabricate a citation. If you cannot find a source, say the claim is unverified.
- Distinguish clearly between what the sources state and your own synthesis or inference.
- When sources disagree, present the disagreement rather than silently picking one.
- End substantial answers with a short "Sources" list de-duplicated by URL.
